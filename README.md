# 📌 About the Project

This project is a **personalized birthday surprise webpage** that I created for my sister's birthday.

Instead of sending a simple birthday message, I wanted to make the celebration more memorable by designing an interactive webpage with animations, music, confetti, and fun surprises.

The webpage includes a virtual birthday cake, a heartfelt birthday letter, interactive gift cards, celebration effects, and a birthday melody, making it feel like a small digital birthday celebration.

---

# ✨ Features

- 🎂 Interactive birthday cake
- 🕯️ Blow out the candle by clicking or using the microphone
- 🎵 Happy Birthday melody generated using the Web Audio API
- 🎊 Beautiful confetti celebration
- 💌 Personalized birthday letter
- 🎁 Interactive gift cards with hidden wishes
- ✨ Floating animated background particles
- 🎨 Modern Glassmorphism user interface
- 📱 Fully responsive design
- 💖 Smooth animations and transitions

# 🎉 How It Works

When the webpage opens:

- A beautiful birthday greeting is displayed.
- Background particles create a festive atmosphere.
- Confetti celebrates the occasion.
- A birthday cake appears with a glowing candle.
- A heartfelt birthday message is shown.
- Interactive gift boxes reveal hidden wishes.

Users can interact with different elements to make the celebration more enjoyable.

---

# 🎂 Interactive Features

## 🕯️ Birthday Cake

The birthday cake is fully interactive.

Users can:

- Click the candle to blow it out.
- Blow into the microphone (if supported) to extinguish the candle.

Once the candle is blown out:

- The flame disappears.
- A birthday wish message is displayed.
- Confetti is launched.
- The Happy Birthday melody starts playing.

---

## 🎵 Birthday Melody

Instead of using an audio file, the project generates the **Happy Birthday** tune using the **Web Audio API**.

This creates the melody directly in the browser without requiring any external music files.

---

## 🎊 Confetti Celebration

Confetti animations are triggered:

- When the page loads
- When the candle is blown out
- When the Celebrate button is clicked

This creates a fun birthday celebration effect.

---

## 🎁 Gift Cards

Three interactive gift cards are included.

Each card flips when clicked to reveal a hidden birthday wish.

---

## 💌 Birthday Letter

A personalized birthday letter expresses love, blessings, and best wishes for the birthday celebration.
